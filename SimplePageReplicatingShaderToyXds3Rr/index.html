Simple page attempting to replicate essential behavior from https://www.shadertoy.com/view/Xds3Rr.

Essential notes from the actual page:

  • The HTMLAudioElement is instantiated in ShaderToy's effect.js via document.createElement('audio'), line 1524.
  • Its src is set up at line 1627 in the same file.
  • Its initial call to play() is made inside the canPlay event listener at line 1583.
  • Afterward, when toggling the play/pause button on the page, calls to HTMLAudioElement.play() are made inside EffectPass.prototype.ResumeInput in the same file, line 660.
  • The event handler for the play/pause button is in pgWatch.js, line 381.

As far as I can tell, the MEI score does not increase when toggling the play/pause button on the site, even though those calls to play() happen inside an event handler with a user gesture.
